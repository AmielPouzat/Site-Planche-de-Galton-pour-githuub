Jeu du coffre

Le coffre choisit un code entre 0 et 7. À toi de retrouver la bonne combinaison avec trois interrupteurs : 1, 2 et 4. Les couleurs suivent le mélange rouge, vert, bleu.

Code : 0
Sélection 000 = 0

Principe

Les trois interrupteurs représentent les puissances 1, 2 et 4. Leur somme donne un nombre de 0 à 7, exactement comme trois bits. Les options de difficulté retirent les indices petit à petit.
