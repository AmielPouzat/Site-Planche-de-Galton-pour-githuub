Générateur déterministe

Portage JavaScript fidèle du script R original. Tu peux utiliser la fonction brute generateur(m, l, a), ou le générateur composé generator2(n) qui produit exactement n valeurs normalisées entre 0 et 1.

Note

Ce n'est pas un générateur cryptographique. Le mode `generator2` suit la construction du script R ; le mode `generateur` donne une version pratique qui continue jusqu'à obtenir le nombre de valeurs demandé. L'indexation temporelle décale seulement le point de départ : elle donne une sensation d'aléatoire, tout en gardant une suite déterministe. `var. pop.` divise par n ; `var. corr.` divise par n - 1, comme `var()` dans R.
